# Insertion Sort

## Idea
Cards arrange karne jaisa
(Left part already sorted)

## Example
Input: [4, 3, 2, 1]  
Output: [1, 2, 3, 4]

## Time Complexity
- Best: O(n)
- Avg / Worst: O(nÂ²)

## Space Complexity
- O(1)

## Stable

## In-place
Yes

## When to Use
- Small dataset
- Nearly sorted data
